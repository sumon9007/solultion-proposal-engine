Alias command for approval workflows.

## Instructions

1. Determine whether the user is approving a single proposal or a modular package
2. Apply `docs/approval-instructions.md`
3. Use `.claude/commands/approve-package.md` for package approval
4. Use `.claude/commands/approve-draft.md` for single-proposal approval
5. Never approve content that still contains placeholders, AI notes, or unresolved review comments
